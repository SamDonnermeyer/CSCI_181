Description:

In this project, you will find your own dataset, perform a data mining analysis on it, and present your dataset, methods, and results in a written report.

About the Data:

The data that are used in this project came from an open-source data repository, Kaggle. The data are a collection of images of written Arabic characters. These images are in the format of 32x32 pixels. The Arabic alphabet contains 28 characters, from alif to yeh. The Arabic alphabet is a cursive based language, where each letter has different forms based on the location it is placed within a word. There is an isolated form (standalone), initial (first letter of a word), medial (middle of the word), and final (last letter of the word) form of each letter. The dataset only contains letters of the isolated form – making a model better able to classify unknown characters.

About the Model:

In this project, I created a convolutional neural network (CNN) – a deep learning algorithm which takes in input images and assigns importance to various aspects of the image (features). These features are then utilized within the CNN when generating classifications. The basic construction of a CNN is based on neurons in the brain that respond to visual stimuli. This makes CNNs an ideal model for image classification – and in this case classifying Arabic characters.
